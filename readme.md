# NeuraLexico

NeuraLexico is the core codebase for a semantic-first cognitive tooling system
designed to support Addvar-style interaction models (memory tagging, ritual
invocation, RAG pipelines, CLI executive mode, etc.).

### Current Status
✅ Repo initialized  
✅ First utility script added (`pytils/directoroot.py`)  
⏳ Core modules not yet implemented  

### Purpose of This Repo
This repo will contain:
- The Addvar memory engine
- Semantic labeling + journaling pipelines
- CLI interface and invocation protocols
- Future integrations (Porcupine, Faster-Whisper, Ollama, Piper-TTS, FAISS, etc.)
